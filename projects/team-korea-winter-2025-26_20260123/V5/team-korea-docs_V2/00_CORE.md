# 00_CORE.md
**Team Korea Winter Dashboard 2025-26**  
**프로젝트 개요 및 핵심 정보**

---

## 🎯 프로젝트 목표

**대한민국 동계 국가대표 선수단 통합 대시보드**

- 43명 선수 실시간 데이터 시각화
- FIS 공식 데이터 기반 성적 추적
- 2026 밀라노-코르티나 동계올림픽 대비

---

## 📊 프로젝트 범위

### 대상 종목 (7개)

1. **Alpine Skiing** (알파인 스키)
2. **Ski Cross** (스키 크로스)
3. **Freestyle - Moguls** (프리스타일 - 모굴)
4. **Freestyle - Park & Pipe** (프리스타일 - 파크앤파이프)
5. **Snowboard - Park & Pipe** (스노보드 - 파크앤파이프)
6. **Snowboard Cross** (스노보드 크로스)
7. **Snowboard Alpine** (스노보드 알파인)

### 주요 기능

**페이지 1: 메인 대시보드**
- 올림픽 카운트다운
- 종목별 선수 분포 (도넛 차트)
- 팀별 현황 (바 차트)
- 연령 분포 (히스토그램)
- 시즌 활동 타임라인
- 최근 메달 현황
- 이번 달의 선수 스포트라이트

**페이지 2: 최신 결과**
- 종목/날짜/선수별 필터
- 경기 결과 테이블
- 선수 상세 정보 모달

**페이지 3: 선수 프로필**
- 선수 상세 정보
- 최근 경기 실적 


**페이지 하단: 링크허브**
- 대한스키협회
- FIS 공식 사이트
- 대한체육회
- 올림픽 공식 페이지

## 🛠️ 기술 스택

### 프론트엔드

- **Framework**: React + TypeScript
- **Build**: Vite
- **Styling**: Tailwind CSS + Google Stitch
- **Charts**: Chart.js / Recharts
- **State**: React Context API

### 데이터 파이프라인

- **Scraping**: Python (requests, BeautifulSoup)
- **Storage**: JSON 파일
- **Scheduler**: cron (매주 수요일 오전 9시)

### 배포

- **Hosting**: Netlify
- **CI/CD**: GitHub Actions

## 🎯 성공 지표

1. **데이터 정확도**: 100% FIS 공식 데이터 일치
2. **업데이트 주기**: 주 1회 자동 업데이트
3. **성능**: 페이지 로딩 < 2초
4. **반응형**: 모바일/태블릿/데스크톱 완벽 지원
5. **접근성**: WCAG 2.1 AA 준수
