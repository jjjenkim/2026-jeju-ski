# 🗂️ Obsidian PARA Organizer - 빠른 시작 가이드

## 📋 3단계로 시작하기

### 1️⃣ 설정 파일 수정

`config.txt` 파일을 열어서 볼트 경로만 수정하세요:

```txt
VAULT_PATH=/Users/jenkim/Library/Mobile Documents/iCloud~md~obsidian/Documents/Mac book 옵시디언
```

> ⚠️ **주의**: 이 경로가 정확해야 합니다!

### 2️⃣ 실행

**Mac 사용자:**
1. `실행.command` 파일을 찾으세요
2. 더블클릭하세요
3. 완료!

**터미널 사용자:**
```bash
cd Obsidian_PARA_Organizer
python3 para_organizer.py
```

### 3️⃣ 확인

Obsidian을 열고 `PARA_System` 폴더를 확인하세요!

---

## 🎯 무엇이 생성되나요?

### 폴더 구조
```
PARA_System/
├── 000 Meta/
│   └── Dashboard.md       ← 여기서 시작!
├── 100 Projects/          ← 프로젝트
├── 200 Areas/             ← 업무/개인/건강/재무
├── 300 Resources/         ← 지식 저장소
└── 400 Archive/           ← 완료된 항목
```

### 자동 생성 파일
- ✅ **Dashboard.md** - 전체 현황 대시보드
- ✅ **MOC 파일들** - 카테고리별 지식 지도
- ✅ **백업** - 실행 전 자동 백업

---

## 🏷️ 파일 분류 방법

### 태그 추가하면 자동 분류!

**프로젝트로 분류:**
```markdown
---
tags: [project]
status: active
---
# 내 프로젝트
```

**업무로 분류:**
```markdown
---
tags: [work]
---
# 업무 노트
```

**독서 노트로 분류:**
```markdown
---
tags: [book]
source: "책 제목"
---
# 독서 노트
```

---

## ⚙️ 기본 설정 (그대로 사용해도 OK!)

```txt
CREATE_BACKUP=true         # 백업 자동 생성
AUTO_MOVE=true             # 파일 자동 이동
INBOX_DAYS=7               # 최근 7일 파일은 Inbox로
ARCHIVE_DAYS=90            # 90일 이상 안 쓴 파일은 Archive로
```

## 💡 처음 사용자를 위한 팁

### 1. 백업 확인
실행 후 볼트에 `_backup_YYYYMMDD_HHMM/` 폴더가 생성됩니다.

### 2. Dashboard 확인
`PARA_System/000 Meta/Dashboard.md`를 열어보세요!

### 3. 분류 확인
파일들이 적절한 폴더로 이동했는지 확인하세요.

### 4. 잘못 분류된 파일은?
수동으로 이동하거나 태그를 추가 후 재실행하세요.

## ❓ 자주 묻는 질문

A: 아니요! AI를 사용하지 않아 완전 무료입니다.
