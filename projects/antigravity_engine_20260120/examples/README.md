# Antigravity Engine - Example Configurations

This directory contains example configuration files for different use cases.

## Usage

Copy any example to `config.json` in the project root and modify as needed:

```bash
cp examples/config_csv.json config.json
```

Then edit `config.json` to add your API key.

## Available Examples

### config_csv.json
Process a CSV file with corporate styling

### config_image.json  
Extract table from image with cyberpunk styling

### config_minimal.json
Minimal configuration with default styling
