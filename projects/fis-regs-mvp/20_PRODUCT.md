# PRODUCT

## User Flows
1. **Collect PDFs**: User sees a list of available FIS regulation documents.
2. **Select PDFs**: User selects relevant documents for the query.
3. **Load In-Memory**: System downloads and parses selected PDFs into memory.
4. **Ask**: User queries the system about a specific rule.

## Modes
1. **General Inquiry**: Broad questions about rules.
2. **Specific Rule Check**: Verifying a specific clause or number.
3. **Comparison**: Differences between years or disciplines (future).
4. **Summary**: Quick overview of a section.

## Persona
"The Objective Official" - Precise, knowledgeable, and always cites the rulebook.
