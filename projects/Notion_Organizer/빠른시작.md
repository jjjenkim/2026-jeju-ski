# Notion Organizer - 빠른 시작 가이드

## 🚀 5분 안에 시작하기

### 1단계: 라이브러리 설치 (1분)
```bash
pip3 install notion-client
```

### 2단계: Notion Integration 만들기 (2분)

1. **https://www.notion.so/my-integrations** 접속
2. **"+ New integration"** 클릭
3. 이름: `Workspace Organizer` 입력
4. **Submit** 클릭
5. **Internal Integration Token** 복사 (ntn_으로 시작하는 긴 문자열)

### 3단계: 부모 페이지 만들기 (1분)

1. Notion에서 새 페이지 생성
2. 제목: `2026_Notion_Index`
3. 페이지 우측 상단 **"..."** → **"Connections"**
4. 방금 만든 Integration 선택
5. 페이지 URL 복사
   - 예: `https://www.notion.so/workspace/2026-Notion-Index-abc123def456`
   - 필요한 ID: `abc123def456` (마지막 32자리)

### 4단계: 설정 파일 수정 (1분)

`config.txt` 파일을 열고:

```txt
NOTION_TOKEN=ntn_여기에_복사한_토큰_붙여넣기
OUTPUT_PAGE_ID=여기에_페이지_ID_붙여넣기
```

### 5단계: 실행! (즉시)

**방법 A: 더블클릭**
- `실행.command` 더블클릭
- (처음에는 우클릭 > "열기")

**방법 B: 터미널**
```bash
python3 notion_organizer.py
```

## ✅ 완료!

Notion의 `2026_Notion_Index` 페이지 아래에 7개 리포트가 생성됩니다!

## 📸 스크린샷 가이드

### Integration Token 찾기
```
https://www.notion.so/my-integrations
↓
+ New integration
↓
이름 입력 → Submit
↓
Internal Integration Token 복사
(ntn_로 시작하는 긴 문자열)
```

### 페이지 ID 찾기
```
Notion 페이지 URL:
https://www.notion.so/workspace/페이지이름-abc123def456
                                        ↑
                                    이 부분 복사
```

### Connection 연결
```
페이지 우측 상단 "..." 클릭
↓
Connections
↓
Integration 이름 선택
↓
✅ 연결 완료
```

## ⚠️ 자주 하는 실수

### ❌ Integration을 페이지에 연결 안 함
→ 페이지 설정에서 Connection 확인!

### ❌ 페이지 ID가 아닌 전체 URL 복사
→ 마지막 32자리만 복사하세요!

### ❌ config.txt에 따옴표 추가
→ 따옴표 없이 그대로 붙여넣기!

```txt
# ❌ 잘못된 예
NOTION_TOKEN="ntn_xxx"

# ✅ 올바른 예
NOTION_TOKEN=ntn_xxx
```
