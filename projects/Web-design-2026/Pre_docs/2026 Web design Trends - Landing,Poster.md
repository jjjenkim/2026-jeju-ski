## 1. 히어로 원샷 랜딩 (No-Scroll Hero)

**컨셉**

- 사용자가 스크롤 안 해도 핵심 오퍼·비주얼·CTA가 1스크린 안에 모두 보이도록 설계합니다.graphicdesignjunction+1
    
- 강한 헤드라인, 짧은 서브카피, 1~2개의 CTA 버튼, 제품/서비스를 직관적으로 보여주는 비주얼이 중심입니다.involve+1
    

**추천 컬러 팔레트 (Conversion Focus)**

|용도|색상명|HEX|
|---|---|---|
|배경|Soft Off-White|#F5F1E8|
|헤드라인 텍스트|Almost Black|#111111|
|서브텍스트|Dark Gray|#333333|
|주 CTA|Sunset Orange|#F07C3C|
|보조 CTA|Deep Blue|#0B6B99|

_설명_: 따뜻한 아이보리 배경에 주황 CTA를 두어 시선을 강하게 끌면서도, 블루 보조 CTA로 선택지를 명확하게 분리합니다.hookagency+1

## 2. 다크 모드 + 네온 액센트

**컨셉**

- 어두운 배경에 네온 계열 포인트로 “테크/크리에이티브/뮤직” 느낌을 살리는 포스터·랜딩이 급증하고 있습니다.fiveelements+2
    
- 폰트와 CTA는 접근성을 위해 명도 대비를 극대화하며, 네온은 10~20% 정도만 사용해 피로를 줄입니다.zoho+1
    

**네온 다크 팔레트**

|용도|색상명|HEX|
|---|---|---|
|메인 배경|True Black|#050509|
|카드/섹션|Deep Navy|#0F1419|
|기본 텍스트|Soft White|#F5F5F5|
|주 CTA|Neon Green|#2CFF05|
|링크/포커스|Neon Blue|#00CAFF|
|강조 라벨|Neon Pink|#FF1493|

_적용 팁_: 포스터 타이틀이나 랜딩 히어로 헤드라인에 네온 그라디언트(예: #FF1493 → #00CAFF)를 적용하면 음악 페스티벌, SaaS, 크리에이터 브랜딩에 잘 맞습니다.varundigitalmedia+1

## 3. 볼드 타이포 중심 포스터 (Type-First Poster)

**컨셉**

- 텍스트 자체가 그래픽 요소가 되도록, 초대형 헤드라인·왜곡·커스텀 폰트를 사용하는 트렌드입니다.adobe+2
    
- 이미지 없이도 메시지와 분위기를 충분히 전달하기 위해 자간/행간/곡선/왜곡을 과감히 씁니다.fiveelements+1

|헤드라인|#F2A1A1|연핑크, 강한 대비[[hookagency](https://hookagency.com/blog/website-color-schemes-2020/)]​|
|서브카피|#FFFFFF|순수 흰색|

## 4. 시네마틱 스크롤 랜딩 (Scrollytelling)

**컨셉**
