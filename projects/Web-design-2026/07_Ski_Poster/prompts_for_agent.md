# Agent Builder 실행용 프롬프트

## 사용 방법
1. 아래 프롬프트를 복사
2. Vertex AI Studio 또는 Agent Builder 콘솔에 붙여넣기
3. 또는 터미널에서 curl 명령어 실행

---

## 프롬프트 1: 눈송이 파티 컨셉

```
Create a complete HTML file for a vertical web poster (1080x1920px) for "2026 Jeju Ski Association Competition".

Design Concept: "Snowflake Party" - Bright and cheerful winter festival

Requirements:
- Vertical layout (9:16 ratio)
- Animated snowflakes falling across the screen
- Skiing character moving in zigzag pattern
- Competition info in snowflake-shaped cards

Color Palette:
- Primary: #FF6B9D (Pink)
- Secondary: #00D9FF (Cyan)
- Accent: #FFE66D (Yellow)
- Background: #F0F4FF (Light Blue)

Typography:
- English headline: 'Fredoka One' from Google Fonts
- Korean headline: 'Black Han Sans' from Google Fonts
- Body: 'Poppins' and 'Noto Sans KR'

Content to include:
- Title: "2026 제주스키협회 스키대회"
- Date: "2026년 2월 27일 금요일"
- Events: "모글 / 알파인 (대회전)"
- Slopes: "핑크 / 메가그린"

Animations:
- Snowflakes falling continuously
- Skier character sliding left-right
- Text bouncing on hover
- Smooth transitions

Output: Complete, self-contained HTML file with embedded CSS and JavaScript. No external dependencies except Google Fonts. Make it playful, witty, and appealing to children.
```

## 통합 프롬프트 (3개 한번에)

```
Create 3 different vertical web posters (1080x1920px each) for "2026 Jeju Ski Association Competition". Each should be a complete, self-contained HTML file.

Competition Details:
- Title: "2026 제주스키협회 스키대회"
- Date: "2026년 2월 27일 금요일"
- Events: "모글 / 알파인 (대회전)"
- Slopes: "핑크 / 메가그린"

Concept 1: "Snowflake Party"
- Colors: #FF6B9D, #00D9FF, #FFE66D, #F0F4FF
- Fonts: Fredoka One, Black Han Sans
- Animation: Falling snowflakes, sliding skier

Concept 2: "Retro Ski Club"
- Colors: #FF5722, #4CAF50, #9C27B0, #FFF9C4
- Fonts: Righteous, Jua
- Animation: Rotating badges, bouncing stickers

Concept 3: "Cyber Slope"
- Colors: #2CFF05, #FF1493, #00CAFF, #1A1A2E
- Fonts: Baloo 2, Black Han Sans
- Animation: Neon glow, glitch effects

Requirements for all:
- Vertical 9:16 ratio
- Playful, witty, appealing to children
- Animated elements (rolling, moving, bouncing)
- Use Google Fonts only
- Complete HTML with embedded CSS/JS
- No external dependencies

Output: 3 separate HTML files, each fully functional and ready to open in browser.
```
